外积、线性变换的定准，以及行列式

RoseL

我们或许曾经介绍过元积 (a0,…,an')⁡(b0,…,bn')=a0⁢b0+⋯+an'⁢bn' （也介绍过，自然数的导元 n' 是指它前面一个数），并用它介绍过线性无关向量组所张成的线空间，以及该空间中任何向量的坐标表示。现在，我们讨论如下这种运算，它叫做外积：

当它写成二元运算 v∧u 的形式时，满足

  • 左线性律 (a⁢u+v)∧w=a(u∧w)+v∧w ；

  • 反交换律 v∧u=-u∧v ；

  • 结合律 u∧(v∧w)=(u∧v)∧w。

至于外积到底算出了什么、怎么表示，这不在本文探讨范围内。

定理 1. 线性律可以生效于 n 元外积式的任意一元上，即

v0∧⋯∧(k⁢p+q)∧⋯∧vn'=k⁢(v0∧⋯∧p∧⋯∧vn')+v0∧⋯∧q∧⋯∧vn'

证明. 利用结合律，将不含 p,q 的部分记作表达式 A,B 并甩到后面，使左边完成关键的展开。

v0∧⋯∧(c⁢p+q)∧⋯∧vn' (代换) = (A∧(c⁢p+q))∧B (反交换律) = (-(c⁢p+q)∧A)∧B (线性) = -c⁢((p∧A)∧B)-(q∧A)∧B (反交换律) = c⁢((A∧p)∧B)+((A∧q)∧B) (反代换) = c⁢(v0∧⋯∧p∧⋯∧vn')+v0∧⋯∧q∧⋯∧vn'

□

注记. 结合律的重要性就在于使表达式变形不需要像冒泡一样逐一交换，可以直接到达指定位置。

记法 2. 记将第 i 元乘到 k 倍为伸缩变换 Di(k) 。设向量组 V=(v0,…,vn') ，那么

⋀Di(k)⁢V = k⁢⋀V

定理 3. 反交换律可以生效于 n 元外积式的任意两元上，即

v0∧⋯∧p∧⋯∧q∧⋯∧vn'=-v0∧⋯∧q∧⋯∧p∧⋯∧vn'

证明. 利用结合律，将不含 p,q 的部分记作表达式 A,B,C ，把它们甩到后面，使左边完成关键的交换，使得所有的反交换律都能配合左线性律使用来提取负号。

v0∧⋯∧p∧⋯∧q∧⋯∧vn' (代换) = (((A∧p)∧B)∧q)∧C (反交换律×2) = (q∧((p∧A)∧B))∧C (结合律×2) = (((q∧p)∧A)∧B)∧C ⁢⋆⁡(反交换律) = -(((p∧q)∧A)∧B)∧C (结合律×2) = -(p∧((q∧A)∧B))∧C (反交换律×2) = -(((A∧q)∧B)∧p)∧C (反代换) = -v0∧⋯∧q∧⋯∧p∧⋯∧vn'

□

注记. 请一定要记住，代换前的结构按照KISS (keep it simple straightforward) 原则设计，最简单直接的那个其实就是最好用的。这里也可以用结合律把表达式代换成 A∧((p∧(B∧q))∧C)，看上去好像只需要把最内层和最外层的表达式甩到后面挺方便，但如果线性律只有左边能用，就反而导致负号不太方便处理。

记法 4. 记将向量组的第 i,j 两元对换为镜像变换 Pi,j 。设向量组 V=(v0,…,vn') ，那么

⋀Pi,j⁢V = -⋀V

推论 5. 外积还具有示零性。n 元外积式中只要有一元为零，或有两元相同，则结果为零。

v0∧⋯∧0∧⋯∧vn' = 0 v0∧⋯∧p∧⋯∧p∧⋯∧vn' = 0

证明. 对零元取负并把负号提出，得

X0 = v0∧⋯∧(-0)∧⋯∧vn' = -(v0∧⋯∧0∧⋯∧vn')=-X0

X0=-X0 ，结果只能是 X0=0 。

把相等元素交换以凭空产生负号，得

Xp⁡p = v0∧⋯∧p∧⋯∧p∧⋯∧vn' = -(v0∧⋯∧p∧⋯∧p∧⋯∧vn')=-Xp⁡p

Xp⁡p=-Xp⁡p ，结果只能是 Xp⁡p=0 。□

注记. 两元相同的示零和两元对换的变号是等价的。我们都知道，交换两个变量的值可以通过这样的算法：`a=a+b; b=-b+a; a=a-b`。已知示零性就可以实现的过程，叫做错切，比如

p∧q=(p+q)∧q=-(p+q)∧p=-q∧p

定理 6. 记将第 i 元上加入第 j 元的 k 倍为错切变换 Ei,j(k) 。设向量组 V=(v0,…,vn') ，那么

⋀Ei,j(k)⁢V = ⋀V

证明. 由示零性，

⋀Ei,j(k)⁢V = v0∧⋯∧(vi+k⁢vj)∧⋯∧vj∧⋯∧vn' = v0∧⋯∧vi∧⋯∧vj∧⋯∧vn'+k⁢(v0∧⋯∧vj∧⋯∧vj∧⋯∧vn') = v0∧⋯∧vi∧⋯∧vj∧⋯∧vn'+0=⋀V

□

定义 7. 初等变换只有伸缩、镜像、错切三种。

对一组向量做初等变换前后，总外积是共线的。比例系数和这个向量组的具体取值无关，而是由初等变换本身确定的。于是比例系数称为初等变换的定准，有

|Di(k)|=k⁡|Pi,j|=-1|Ei,j(k)|=1

我们十分清楚线性映射和矩阵是同构的，那么高斯消元法暗示，任何线性变换总能表示为一系列初等变换的接连作用。把这些初等变换的定准连乘起来，就是线性变换的定准。

把线性变换的方阵表示两边的框线改写成竖线，让它成为一个算式，叫做行列式。行列式的值就是这个线性变换或这个矩阵的定准。

推论 8. 上三角行列式和下三角行列式的值为其主对角线元素的总乘积。

推论 9. 非零 n 元组线性相关，则它们的总外积为零。

证明. 设向量组 V=(v0,…,vn') 线性相关，则存在一组数 (c0,…,cn'') 使得

vn'=∑i=0n''ci⁢vi

于是

⋀V = v0∧⋯∧vn-2∧vn' = v0∧⋯∧vn-2∧∑i=0n''ci⁢vi = ∑i=0n''ci⁢(v0∧⋯∧vn''∧vi)=0

□

算法 1

将一组自然数 (0,…,n') 数次任选两个数交换位置，得到排列 (i0,…,in') ，可以包括冗余的交换。如果交换的次数是奇数，那么记 δ(i0,…,in')=-1 ，否则交换的次数是偶数，记 δ(i0,…,in')=1 。

于是设一个 n 维线性空间 U 上有一个线性变换 T ，它将这个线性空间上的一组基 Ξ=(ξ0,…,ξn') 的各向量映射为 T[Ξ]=(T(ξ0),…,T(ξn')) 。但因为 Ξ 是基，所以 U 中的向量 T(ξi) 一定能用 Ξ 线性表示，坐标分别是 Ci 。

让 n 个计数变量同时在这组自然数上遍历。这些变量一旦有两个取到相同值，那么它们指到的向量的总外积 (ξi0∧⋯∧ξin') 就会示零。不示零，就需要这些计数变量不重复地取遍这组自然数，但这组计数变量和这组数一样多，于是不重不漏，得到一个排列。所有可能的不示零项正是通过这么遍历全排列来枚举的，然后发现它们共线。

⋀T[Ξ] = T(ξ0)∧⋯∧T(ξn') (坐标表示) = (C0⁢Ξ)∧⋯∧(Cn'⁢Ξ) (线性展开) = ∑i0=0n'⋯⁢∑in'=0n'(C0,i0⁢⋯⁡Cn',in')⁢(ξi0∧⋯∧ξin') (识别示零) = ∑(i0,…,in')(C0,i0⁢⋯⁡Cn',in')⁢(ξi0∧⋯∧ξin')+0+⋯+0 (识别交换) = ∑(i0,…,in')(C0,i0⁢⋯⁡Cn',in')⁢δ(i0,…,in')⁢(ξ0∧⋯∧ξn') (提取单位) = (∑(i0,…,in')(C0,i0⁢⋯⁡Cn',in')⁢δ(i0,…,in'))⁢(ξ0∧⋯∧ξn')

这个算法本身并不重要，也绝不实用，却是以一种独立的方法知道，对一组向量做线性变换前后，外积是共线的，比例系数就是线性变换的定准，有 ⋀T[Ξ]=|T|⁢⋀Ξ 。
